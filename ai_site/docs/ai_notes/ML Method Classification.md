- Nearest Neighbor
- Neural Networks

### Unsupervised Learning
> Discover an internal representation from input data only.

#### Clustering Algorithms
- k-Means
- k-Medoids
- Fuzzy C-Means
- Hierarchical Clustering
- Gaussian Mixture Models
- Neural Networks
- Hidden Markov Models
